# BACKEND do TEAhub

Repositório para o Backend do TEAhub.

---

Passos para instalar:

```python
#instalar venv
py -m venv .venv

#ativar venv
./.venv/Scripts/Activate.ps1

#instalar dependências
pip install -r requirements.txt

#iniciar API
uvicorn app.main:app --reload
```
